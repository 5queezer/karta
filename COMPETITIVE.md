## Key Dimensions for Karta

### Where we lead
- **Auditability**: Only system (with A-MEM) where every note, link, evolution, and dream is inspectable JSON. Honcho/EverMemOS are opaque.
- **Operational cost**: Embedded Rust library, zero infrastructure. Every competitor except A-MEM requires Docker or hosted services.
- **Dream diversity**: 5 typed dream modes with cross-cluster sliding windows. Honcho has similar but opaque. EverMemOS has consolidation only.
- **Write-time organization**: Retroactive evolution + semantic linking + turn_index ordering. Only EverMemOS matches with event-boundary segmentation.

### Where we trail
- **BEAM score**: 57.8% vs Hindsight 75% (17.2pp gap, new leader), Honcho 63.0% (5.2pp gap). Primary driver: event ordering (37%) and knowledge update (38%). Hindsight has overtaken Honcho as BEAM leader.
- **LOCOMO**: Not benchmarked yet (run hung on query phase, needs debugging). EverMemOS 93.05%, Hindsight 92% (AMB), MemU 92.09%, MemMachine 91.69%, Honcho 89.9%, Memori 81.95%, Engram 80.0%, Letta 74.0%.
- **LongMemEval**: 50.0% on oracle split (50 questions, all temporal-reasoning). Not a balanced score, needs full run across all 6 question types. OMEGA 95.4% (#1 leaderboard), Mastra 94.87% (gpt-5-mini), Hindsight 94.6% (AMB), MemMachine 93.0%, MemPalace 96.6% R@5 (retrieval recall, not answer quality).
- **Forgetting**: Phase 3 planned but not built. EverMemOS and Memori have this as core principles (intelligent decay).
- **Episode segmentation**: Data model ready but not wired. EverMemOS has this working and it's their key differentiator for LOCOMO.
- **Context efficiency**: Memori achieves 81.95% LoCoMo at 4.98% the cost of full context (avg 1,294 tokens/query). Engram achieves 80% LOCOMO at 96.6% fewer tokens. Mastra achieves 5-40x compression with high prompt cache hits. Cost-efficiency is becoming a key differentiator.
- **Letta baseline threat**: Letta Filesystem (simple file storage + grep) scores 74% on LOCOMO. If specialized memory systems don't substantially beat this, the value proposition is weakened. Karta needs to demonstrate clear superiority before publishing LOCOMO numbers.
- **Hindsight benchmark dominance**: Hindsight (Vectorize) has emerged as the benchmark leader across BEAM, LOCOMO, and LongMemEval simultaneously (Apr 2026). Their open-source Python library + hosted API is a direct competitive threat.
